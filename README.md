# ICMS –  Industrial Cooling and Monitoring System

Desktop app built with Electron, Vite, and React.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Starts Electron with the renderer served by Vite (hot reload).

## Build

```bash
npm run build
```

Output: `out/main`, `out/preload`, `out/renderer`.

## Run built app

```bash
npm start
```
